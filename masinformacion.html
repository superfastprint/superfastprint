La máquina de Autoservicio (Super Fastprint) de impresión es similar a un cajero automático, cuenta con un software especializado para dispensar documentos impresos.

Imprimir documentos será tan sencillo como obtener un café en una máquina o sacar dinero de un cajero automático.

Se trata de una impresora pública de documentos con soporte digital, bastará con colocar el dispositivo USB o conectarse vía aplicación desde un dispositivo android, marcar las opciones de impresión, insertar el monto requerido en monedas y dispensar las impresiones solicitadas sin asistencia de nadie. Documentos en formatos en PDF, Word, Excel, Power Point, entre otros, son los que este novedoso sistema acepta para que puedan ser impresos.
